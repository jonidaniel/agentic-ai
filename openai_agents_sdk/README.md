# OpenAI Agents SDK

There are two separate projects here:

1. `simple_jokester_agent.ipynb`

- An individual agent capable of replying with a joke

2. `deep_research_recreated.ipynb`

- OpenAI's Deep Research re-created using OpenAI Agents SDK

- A multi-agent orchestration following a _prompt chaining_ architecture

- Consists of four agents assigned to separate tasks:

1. Planner agent
2. Searcher agent
3. Writer agent
4. Pusher agent
