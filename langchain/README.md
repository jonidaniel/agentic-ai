# LangChain

There are four simple LangChain model demos:

`langchainchatmodels_initchatmodel_gpt41.ipynb`

- Imports `init_chat_model` from `langchain.chat_models`
- Uses `gpt-4.1` as LLM

`langchainopenai_chatopenai_gpt5.ipynb`

- Imports `ChatOpenAI` from `langchain_openai`
- Uses `gpt-5-nano` as LLM

`langchainopenai_openai_gpt35.ipynb`

- Imports `OpenAI` from `langchain_openai`
- Uses `gpt-3.5-turbo-instruct` as LLM

`openai_openai_gpt5.ipynb`

- Imports `OpenAI` from `openai`
- Uses `gpt-5-nano` as LLM
